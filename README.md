# 🎓 Student Registration System

The **Student Registration System** is a menu-driven console application developed in **Java** that allows users to manage student records efficiently using **MySQL** and **JDBC**. The system provides functionalities to register new students, view all student records, update student information, delete student records, and search students through an interactive console interface.

This project was developed to strengthen my understanding of **Core Java, JDBC, MySQL, Object-Oriented Programming (OOP), and Database Connectivity**, while gaining practical experience in building a real-world student management application with secure and efficient database operations.

# 🚀 Getting Started

## 📥 Clone the Repository

```bash
git clone https://github.com/ParneetKaur-coder/StudentRegistrationSystem.git
```

## 📂 Open the Project

Open the project using any Java IDE:

- 💡 IntelliJ IDEA
- 🌙 Eclipse
- 🟢 NetBeans
- 💻 VS Code

## 🗄️ Configure MySQL Database

1. Install MySQL Server.
2. Create the database using `DatabaseSetup.sql`.
3. Update your MySQL username and password in `DBConnection.java`.
4. Add the MySQL JDBC Connector (`mysql-connector-j.jar`) to your project.

## ⚙️ Compile the Project

```bash
javac *.java
```

## ▶️ Run the Application

```bash
java Main
```
